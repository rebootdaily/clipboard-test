Clipboard v6.0 - Point-to-Point Production Test

Primary sketch workflow:
1. Tap the first corner.
2. Tap the next corner to indicate direction.
3. Enter or speak the measured wall length.
4. Repeat.
5. Tap within 3 feet of the first corner and answer Close Structure?

Canvas gestures:
- Pinch: zoom
- Two-finger drag: pan
- Mouse wheel: zoom
- Double-tap: fit sketch

Voice input uses the browser Speech Recognition API and is intended for current Microsoft Edge or Chrome.
