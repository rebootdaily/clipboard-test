Clipboard v6.1 Guide Points Beta

New sketch behavior:
- A visible green START point is created at the center of the canvas.
- Eight 90-degree and 45-degree guide points appear around the active corner.
- Tap a guide point for an exact direction, or tap elsewhere for a custom angle.
- Enter wall length by keyboard or microphone.
- Tap near the start after at least three walls to receive the Close Structure prompt.
- Pinch zoom, two-finger pan, mouse-wheel zoom, Fit, Undo, and Redo remain available.

Install by copying these files over the existing clipboard-test folder while preserving the .git folder. Run Start_Clipboard_v4.bat and refresh with Ctrl+F5.
