Clipboard v6.3 Geometry Assist Beta

Changes:
- Gold assist points now align the active corner with any existing sketch corner, not only START.
- Horizontal and vertical projection candidates are generated from prior corners.
- Faint gold construction lines show which existing corner controls each assist point.
- START closure assistance remains available within 45 feet.
- Geometry candidates are limited and prioritized to keep the canvas readable.
- Calculated walls continue to be labeled CALC.

Test workflow:
1. Draw several measured walls.
2. At the active point, look for gold ALIGN points horizontally or vertically aligned with prior corners.
3. Tap a gold point and select Use Calculated or Measure Manually.
