Clipboard v6.4 Smart Sketch Beta

This build preserves Point-to-Point mode and adds an experimental Smart Sketch mode.

Smart Sketch workflow:
1. Select Smart Sketch Beta.
2. Start a stroke on the blue active corner.
3. Draw roughly toward the next corner.
4. Clipboard straightens the stroke and snaps near 0, 45, 90, 135, 180, 225, 270, or 315 degrees.
5. Enter or speak the measured wall length.
6. Repeat.

Point-to-Point, Geometry Assist, calculated closure walls, voice length entry, pinch zoom, two-finger pan, and all existing v6.3 features remain available.

Testing notes:
- Begin each Smart Sketch stroke close to the blue active point.
- A green dashed preview indicates a 45-degree snap.
- A blue dashed preview indicates a custom angle.
- Use two fingers for pan/zoom; one-finger strokes are reserved for Smart Sketch drawing.
