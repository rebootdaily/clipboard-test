Clipboard v7.0 - Digital Field Sketch

The previous Point-to-Point and Smart Sketch user interface has been replaced by a paper-style Field Sketch workspace.

Tools:
- Pencil
- Straight line
- Arrow
- Circle
- Room/area label
- Dimension text
- General note
- Stroke eraser
- Pan, pinch zoom, and +/- zoom
- Undo/redo
- Multiple sketch pages/floors
- Background image import
- PNG export

Analyze Sketch is intentionally reserved for v7.1. Original strokes are stored in inspection state.
