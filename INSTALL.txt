Clipboard v6.4 Smart Sketch Beta

1. Stop the Clipboard local server.
2. Extract this ZIP.
3. Copy its contents into the existing clipboard-test folder.
4. Preserve the existing .git folder.
5. Replace files when prompted.
6. Run Start_Clipboard_v4.bat.
7. Refresh Edge with Ctrl + F5.

The Sketch page should display 6.4 Smart Sketch Beta.
