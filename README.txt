Clipboard v5.0b Sketch Update

- app.js

Keep the existing .git folder and other repository files.

Implemented:
- Measurement-first sketch workflow
- Length plus four-direction wall entry

- Finish first, classify areas afterward
- Back-Out Area polygons for garage, open-to-below, patio, porch, courtyard, etc.

- Multiple structures
- Local browser saving
- Mobile-friendly layout
