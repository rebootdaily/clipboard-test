Clipboard v4.2 - Manual Sketch Release

WHAT IS NEW
- Dedicated Sketch tab added automatically to the app navigation.
- Freehand drawing tool.
- Straight wall/line tool.
- Rectangle tool for footprints, garages, patios, pools, and outbuildings.
- Eraser tool removes individual sketch objects.
- Undo and redo (up to 40 sketch states).
- Zoom in, zoom out, fit/center, and pan.
- Import an assessor sketch, survey, or floor plan as a faded background.
- Sketch is auto-saved with the inspection in browser storage.
- Existing workbook-driven fields, visibility rules, follow-ups, and validation remain intact.

BUILD
1. Keep Clipboard_v4_Master_Base.xlsx in this folder.
2. Run Start_Clipboard_v4.bat on Windows or start_clipboard_v4.command on macOS.
3. The generator creates/updates clipboard_generated.
4. Open the local address shown by the launcher.

SKETCH USE
- Open the Sketch tab.
- Select Freehand, Wall, Rectangle, Eraser, or Pan.
- Import Background accepts an image from the device.
- Undo/Redo applies to drawing changes.
- Clear Drawing keeps the imported background.
- Remove Background keeps the drawing.

VALIDATION
The included workbook validates with Issues: 0.
