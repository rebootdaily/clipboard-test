Clipboard v4.4 - Appraisal Sketch Release

Implemented:
- Separate structures and floors
- Main dwelling versus outbuilding GLA treatment
- Calculated rectangular area sections
- Open-to-below subtractive areas
- Exterior wall lines with dimension labels
- Freeform non-calculated reference lines
- Freehand drawing and annotations
- Copy-floor workflow for similar second floors
- Per-floor property-appraiser sketch backgrounds
- Automatic main GLA and structure/floor area summaries
- Undo, redo, zoom, pan, eraser, and floor clearing

Calculated Area workflow:
Draw a rectangle and enter its measured length and width. Multiple rectangles can be combined for irregular footprints.

Open to Below workflow:
Draw the upper-floor void and enter its dimensions. It is deducted only from that floor.

Outbuildings:
Add a separate structure and choose Cancel when asked whether it is included in main dwelling GLA.
