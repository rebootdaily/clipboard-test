Clipboard v6.2 Closure Assist Beta

New in this revision:
- Gold Closure Assist points appear when the active corner is within 45 feet of START.
- ALIGN points place a generated horizontal or vertical endpoint level with the green START point.
- CLOSE point connects directly to START.
- Tapping a gold point shows the calculated wall length.
- Choose Use Calculated, Measure Manually, or Keep Open.
- Calculated walls are labeled CALC and displayed as a gold dashed overlay.
- Existing 90/45 guide points, voice length entry, zoom, pan, and point-to-point drawing remain available.

Install by extracting over the existing clipboard-test folder while preserving .git, then restart the server and press Ctrl+F5.
